# Mapbox Keyframe Animation

This project is a web application built with Vite, focusing on creating keyframe animations with Mapbox.

## Features

-   **Mapbox Integration**: Utilizes Mapbox for interactive maps.
-   **Keyframe Animations**: Implements keyframe-based animations.
-   **Vite Development Server**: Fast development experience with Vite.

## Getting Started

### Prerequisites

-   Node.js (LTS version recommended)
-   npm or yarn

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/jamie-silca/mapbox-keyframe.git
    cd mapbox-keyframe
    ```
2.  Install dependencies (if a `package.json` is eventually added):
    ```bash
    # npm
    npm install
    # or yarn
    yarn install
    ```

### Running the Development Server

```bash
# npm
npm run dev
# or yarn
yarn dev
```

The application will be accessible at `http://localhost:5173`.

## Project Structure

-   `vite.config.js`: Vite configuration file.
-   `style.css`: Main stylesheet.
-   `index.html`: Main HTML file.
